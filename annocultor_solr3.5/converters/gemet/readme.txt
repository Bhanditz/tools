This is a converter to the GEMET dataset.

The files in the input_source directory are downloaded from http://www.eionet.europa.eu/gemet/
They are distributed under the Creative Commons CC BY 2.5 License, at the time of download on 19.03.2011.
With this file and link we attribute the authors.
Before reusing them in your product, please, check the license terms and attribute the authors as well.

Running the converter:
convert.bat
or (to save log messages to a file)
convert.bat > log.txt

Running conversion report:
report.bat

Conversion report would be saved in doc/Gemet_Gemet/index.html, where tab 'Rules' should be the most interesting one for this conversion.
